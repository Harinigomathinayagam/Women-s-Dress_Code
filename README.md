# StyleHer: A Smart Women’s Dress Code & Shopping Guide

In today’s fast-paced world, choosing the right outfit for the right occasion can often feel overwhelming. To simplify this everyday decision-making process, I developed StyleHer, an interactive web-based application designed to guide women in selecting outfits tailored to their occasion and body type.

# Project Overview

StyleHer is a user-friendly fashion assistant built using HTML, CSS, JavaScript, and Bootstrap principles. The platform provides personalized styling recommendations by combining two essential factors: the event type and the user’s body shape. The goal is to make fashion more accessible, confident, and data-driven.

# Key Features

1. Occasion-Based Selection
Users begin by selecting the type of event they are dressing for, such as:

Casual outings
Office or work environments
Parties and weddings
Date nights
Beach or gym activities
Formal events

This ensures that recommendations are contextually appropriate.

2. Body Type Personalization
The app considers different body types, including:

Hourglass
Pear
Apple
Rectangle
Inverted Triangle

By doing so, it promotes body-positive fashion choices that enhance confidence and comfort.

3. Step-by-Step Interactive UI
The interface is designed as a guided flow:

Step 1: Choose occasion
Step 2: Select body type
Step 3: View tailored recommendations

Navigation controls like “Back” and “Start Over” improve usability and user experience.

4. Minimalist and Elegant Design
The application uses clean typography (Playfair Display and Poppins) and a structured card layout to create a modern, aesthetically pleasing interface.

# Technical Implementation
Frontend: HTML5 for structure, CSS3 for styling, and JavaScript for interactivity
UI Design: Card-based layout with responsive grid system
Logic Handling: Dynamic content rendering based on user input selections
Scalability: Easily extendable to include shopping links, AI recommendations, or user profiles
